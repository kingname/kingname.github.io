参数格式

参数字段

字段名 是否必填 字段类型 意义
TaskID 是 str 业务ID，生成的PDF文件也会使用这个ID作为文件名
insuredTitle 是 str 称呼，只能使用以下几项：先生，小姐，女士，太太，小朋友，博士
insuredLastName 是 str 用户的姓，人工填表单的时候，填VIP的那个地方
insuredSmokingHabit 是 int 用户吸烟习惯，1表示吸烟，0表示不吸烟
insuredFirstName 否 str 用户名，英文大写
insuredAge 是 int 建议书年龄
birthYear 否 int 出生年份
birthMonth 否 int 出生月份
birthDay 否 int 出生日
ChineseFirstName 否 str 中文名（不含姓）
ChineseLastName 否 str 中文姓
insuredCountry 否 str 居住区域，只能填“CN”或者“HK”
insuredActualAge 否 int 用户实际年龄
insuredOccupation 否 str 用户职业编号
currency 否 str 货币单位，只能填”USD”或者“HKD”
basicProductType 是 str 产品类型，只能填以下几个值（不含括号内的值）：all（全部）, Annuity（年金）， Unit Link（投资相连），WL（终身寿险），Term（定期寿险），Medical（医疗保障），Major Illness（危疾保障），DP（伤病保障）
basicPlanCode 是 str 计划书产品ID，例如“HB1CILA”或者“BWSVA”
basicSumIns 是 float 投保额
basicModPrem 是 float 每期保额

注意：basicSumIns与basicModPrem任选一项即可，如果两个参数都写了，爬虫只会使用basicModPrem
