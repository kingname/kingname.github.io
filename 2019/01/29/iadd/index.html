大家经常在一些博客中看到这样的说法：

这里报错了，说明a += b和a = a + b并不是完全等价的。

实际上，这是由于+=会首先调用左边这个对象的__iadd__方法，如果没有__iadd__方法，就会调用__add__方法。但是如果直接使用+号，就会直接调用__add__方法。而对于字符串、数字、浮点数这种不可变对象，他们没有__iadd__方法，所以对他们来说，a += b 与 a = a + b是等价的。
