这样导入了.js文件以后，里面的代码在网页加载完成后就会运行。

但在Node.js生态里面，第三方包一般需要使用npm安装，然后在代码里面通过require导入。最后再用webpack打包编译成能直接在浏览器中运行的JavaScript代码。

如果我找到一个第三方的包，它只提供了npm版本，没有提供直接在浏览器中导入的版本怎么办？
